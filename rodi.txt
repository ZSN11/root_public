git clone https://github.com/redrepublics/root_public
git status - проверка статуса в репозитории
git add . - под версионный контроль (точка отдельно)
git commit - фиксация изменений 
-a — совершит коммит, автоматически индексируя изменения в файлах
проекта. Новые файлы при этом индексироваться не будут! Удаление же файлов
будет учтено.
-m «commit comment» — комментируем коммит прямо из командной строки
вместо текстового редактора.
FILENAME — внесет в индекс и создаст коммит на основе изменений
единственного файла.
git push - публикация изменений на сервере
git log - история изменений
git rm fulename.filetype - удаляет файл из под версионного контроля
git branch --list -список веток в локальном репозитории
git branch --list -v -список последних коммитов в ветках
git checkout testbranch00 - переход в ветку testbranch00
git checkout - переместит на шаг назад (можно будет работать с предыдущей версией файла)
git branch testbranch00 - создаем ветку testbranch00
git push --set-upstream origin testbranch01 - публикация в ветку
git merge - с указанием ветви, мерджим
git pull - получение файлов с удаленного репозитория
